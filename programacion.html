Crear Ambiente Virtual en Python

  • Para crear ambiente:
    • python -m venv .venv
  • Para activar el ambiente creado:
    • .venv\Scripts\activate (en CMD)
    • .venv\Scripts\activate.ps1 (en PowerShell)
  • Para desactivar ambiente:
    • .venv\Scripts\deactivate.bat
