Instalar Control de Versiones Git

  • En Windows 10, descarga instalador de Git (https://git-scm.com/downloads/win), luego haz click en archivo descargado para iniciar la instalación, hacer click en Next en cada paso, dejar la ubicación por defecto y los componentes a instalar.
  • Después de instalado, verifica la misma, abriendo la consola CMD o Git Bash y ejecuta:
  • git --version
  • Deberá salir un mensaje mostrando la versión de Git, ahora se debe configuar el nombre del usuario y el Correo del Usuario
    • git config --global user.name "Tu Nombre"
    • git config --global user.email "tucorreo@example.com"

Crear Repositorio Local en Git

  • git init
  • echo "Repositorio Local" > README.md
  • git add .
  • git commit -m "Primer commit"
  • git status
  • git init (inicializa el repositorio local), echo "mensaje" > README.md (Crea un archivo README, informativo del repositorio), git add . (añade todo los archivos para incluirlos en el próximo commit), git commit -m "mensaje" (pasa los archivos en el repositorio), git status (muestra estatus del repositorio)

    • Actualizar Repositorio Local en Git

  • Ya teniento el repositorio iniciado anteriormente
    • git add .
    • git commit -m "Explicación de cambios"
    • git status
  • git init (inicializa el repositorio local), echo "mensaje" > README.md (Crea un archivo README, informativo del repositorio), git add . (añade todo los archivos para incluirlos en el próximo commit), git commit -m "mensaje" (pasa los archivos en el repositorio), git status (muestra estatus del repositorio)

    • Comandos Git Adicionales

    • Ver todos los Commits realizados en Repositorio Local:
      • git log

    Subir Repositorio Local a GitHub

    • Para subir un repositorio local a GitHub, primero debemos tener una cuenta en la WEB de Github.com y crear un repositorio alli.
    • Después de haber hecho un Commit en el repositorio local, verificamos primero que no haya algo pendiente:
    • git status
    • Si no hay archivos pendientes, podemos conectar el repositorio local con GitHub y subir por primera vez el repositorio:
      • git branch -M main
      • git remote add origin URL_DEL_REPOSITORIO
      • git push -u origin main
    • Para actualizar en GitHub, luego de un Commit local:
    • git push -u origin main

    Clonar un Repositorio Existente en GitHub

    • Ve a GitHub, al repositorio deseado. Presiona el botón <>Code, copia la dirección HTTPS, ubicate en la carpeta local donde quieres colocar el repositorio.
    • Abre la consola (CMD) y ejecuta:
    • git clone https://github.com/USER/REPOS.git
    • Nota: Recuerda que debes tener instalado Git para poder clonar un repositorio existente.
