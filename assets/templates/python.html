Crear Ambiente Virtual en Python

  • Para crear ambiente:
  • python -m venv .venv
  • Para activar el ambiente creado:
  • .venv\Scripts\activate (en CMD)
  • .venv\Scripts\activate.ps1 (en PowerShell)
  • Para desactivar ambiente:
  • .venv\Scripts\deactivate.bat

Instalar Paquetes Externos en Python

  • Para instalar los paquetes externos en Python se utiliza el gestor de paquetes pip, primero verificamos que lo tenemos instalado:
  • pip --version
  • (Si nos sale un número de versión esta instalado, de lo contrario se tendrá que instalar)
  • Para instalar pip:
  • python get-pip.py
  • Para instalar paquetes externos:
  • pip install nombre_paquete

Ver Paquetes Instalados en Entorno Python

  • Para ver una lista de los paquetes instalados:
  • pip freeze
  • Para respaldar toddos estos paquetes instalados en el entorno:
  • pip freeze > requirements.txt
  • Luego se pueden instalar en otro entorno:
  • pip install -r requirements.txt
