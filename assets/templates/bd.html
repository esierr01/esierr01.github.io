Instalar e Iniciar MongoDB

  • Se descarga el instalador:
  • Se recomienda instalar la versión 5 que incluye el Shell
  • En la instalación seleccionar Completa y desmarcar opción de encender el motor de bd al iniciar Windows, ya que esto recargará la memoria si no se a utilizar constantemente)
  • Luego de instalado, hay que hacer dos configuraciones:
  • Incluir en el path el directorio ....\bin de MongoDB
  • Crear un directorio en la raiz C:\data\db
  • Para verificar la instalación, ejecutar en consola, deberá aparecer la versión de MongoDB:
  • mongod --version
  • Ejecutar en consola y se iniciara el motor de bases de datos MongoDB
  • mongod
  • El motor de MongoDB se quedará ejecutando en un bucle, no cerrar y minimizar
  • Ahora para ejecutar el Shell de comandos de MongoDB, se abrirá una consola CMD y se ejecutará:
  • mongo
  • Se abrirá el Shell donde puede introducir los comandos de MongoDB
