Instalar e Iniciar MongoDB

  • Se descarga el instalador:
    • Se recomienda instalar la versión 5 que incluye el Shell

      • (En la instalación seleccionar Completa y desmarcar opción de encender el motos de bd al iniciar Windows, ya que esto recargará la memoria si no se a utilizar constantemente)

  • Luego de instalado, hay que hacer dos configuraciones:
    • Incluir en el path el directorio ....\bin de MongoDB
    • Crear un directorio en la raiz C:\data\db
  • Para verificar la instalación, ejecutar en consola:
    • mongod --version

      • (se deve mostrar la versión de MongoDB instalada)

    • ejecutar mongod

      • (Se iniciara el motor de base de datos de MongoDB y quedará en un bucle, No Cerrar, minimizar)

  • Para ejecutar el Shell:
    • Abrir otra ventana del CMD y escribir el comando:
    • mongo

      • (Se abrirá el Shell donde puede introducir los comandos de MongoDB)
